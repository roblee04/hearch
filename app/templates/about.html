About Hearch

A serendipity-focused search engine for curious minds

What is Hearch?

Hearch is an experimental search engine inspired by @alexmolas' microsearch and the philosophy of Marginalia Search. It's designed to surface unexpected and interesting content from a curated collection of documents.

Key Features

BM25 Ranking

Industry-standard Okapi BM25 algorithm for text relevance scoring

Metadata Scoring

Marginalia-inspired bonuses and penalties based on document quality signals

AI Query Expansion

LLM-powered generation of alternative search queries for broader discovery

Async Web Crawling

High-performance asynchronous document discovery and indexing

Technology Stack

  • FastAPI - Modern Python web framework
  • BM25 - Okapi ranking algorithm
  • OpenAI API - Query expansion with LLMs
  • Pandas - Data processing and indexing
  • Bootstrap 5 - Responsive UI framework

Philosophy

Unlike commercial search engines that prioritize SEO and advertising, Hearch focuses on serendipitous discovery. It rewards quality signals like link topology, content length, and publication date while penalizing commercial markers like tracking scripts and ad networks.

This is an experimental project exploring alternative approaches to web search and information retrieval.
